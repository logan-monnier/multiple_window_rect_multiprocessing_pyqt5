# multiple_window_rect_multiprocessing_pyqt5

projet qui en se lançant créer 2 fenetres et un carré, le carré se situe au départ en haut à gauche de l'écran d'ordinateur et peut être bouger avec les touches directionnelles.  
Le but du projet est de pouvoir partager l'information du carré entre les 2 fenetres par du multiprocess, permettant donc de croire que le carré peut passer d'une fenetre à l'autre.  
Il est possible de bouger le carré ou les fenetres pour voir les différents changements.
